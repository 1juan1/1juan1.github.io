"HOME" "PROJECT" "RESEARCH PAPER" "TUTORING"

My local baseball stadium finder is very helpful for those fanatic baseball players to locate a stadium. When you click a stadium it shows the name of the stadium which is a link for info on it in Wikipedia. Maybe one of the stadiums on my stadium finder will be the one you may be headed to. This program is pretty straight forward when using it. The language I used to code this was mostly JavaScript but additionally I used Google Maps API.
